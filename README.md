# React To-Do List App

A clean and simple to-do list web app built with React functional components and hooks.

## Features

- Add, edit, delete tasks
- Mark tasks as completed
- Filter by **All / Completed / Pending**
- Task counts (total and completed)
- Persists tasks and theme in `localStorage`
- Light / dark mode toggle
- Responsive and minimal design with smooth task add animations

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the URL shown in the terminal (usually `http://localhost:5173`) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
